# Проект по извлечению навыков из резюме с помощью FastAPI и aiogram

Этот проект представляет собой веб-сервис, который извлекает ключевые навыки из текста резюме, используя библиотеку spaCy для обработки естественного языка. Сервис доступен через API и может быть интегрирован с Telegram-ботом.

## Структура проекта

Проект состоит из двух основных компонентов: FastAPI-приложение для извлечения навыков и aiogram-бот для взаимодействия с пользователями.
```
project_root/
├── app/
│ ├── init.py
│ ├── api.py # Основной код FastAPI приложения
│ ├── models.py # Pydantic-модели для валидации данных
│ └── skills_keywords.py # Список ключевых слов (навыков)
└── bot/
└── main.py # Код для запуска Telegram-бота
```

## Функциональность

### 1. Извлечение навыков

Основная функция `extract_skills(text: str)` принимает на вход текст резюме и возвращает список уникальных навыков, найденных в тексте. Для этого используется:

* **spaCy**: Библиотека для обработки естественного языка.
* **PhraseMatcher**: Компонент spaCy, который эффективно ищет фразы из заранее заданного списка (`SKILLS_KEYWORDS`).
* **Список ключевых слов**: В файле `skills_keywords.py` хранится большой список технологий, языков программирования, фреймворков и других навыков, которые могут встречаться в резюме.

Функция возвращает только уникальные навыки, приведенные к нижнему регистру, что помогает избежать дублирования (например, "Python" и "python" будут считаться одним навыком).

### 2. FastAPI API

Приложение FastAPI предоставляет один эндпоинт:

* **POST `/extract_skills/`**: Принимает JSON-объект с полем `text`, содержащим текст резюме. Возвращает ответ в формате:
  ```json
  {
    "skills": ["python", "sql", "machine learning"]
  }


### 3. Запуск сервиса
В корне проекта находится скрипт для одновременного запуска FastAPI-сервера и Telegram-бота.

* **Запуск сервера:**Используется uvicorn для запуска FastAPI-приложения на порту 8005.
* **Запуск бота:** Вызывается функция start_bot() из модуля bot.main.
* **Параллельный запуск:** Оба процесса запускаются параллельно с помощью asyncio.gather.
* **Для запуска всего проекта необходимо выполнить команду:**
  ```
  python main.py
  ```
